# SM OS 2.4 — Creación guiada de proyecto

## Nuevo
- Tipo de creación al dar de alta proyecto:
  - Proyecto vacío.
  - Usar plantilla.
  - Creación guiada.
- Vista previa operativa antes de guardar.
- Plantillas base:
  - Consultoría general.
  - Campaña digital.
  - CRM / implementación.
  - Evento / activación.
  - Alianza / patrocinio.
  - Diagnóstico operativo.
  - Seguimiento comercial.
- Creación guiada por necesidades del cliente:
  - Estrategia, ventas, marketing, CRM, operación, eventos, reportes, contenido, patrocinios, documentación y seguimiento comercial.
- El sistema crea automáticamente frentes y tareas base cuando se elige plantilla o creación guiada.

## Nota
- La opción Proyecto vacío conserva el flujo actual.
- Esta versión no modifica proyectos existentes.
