# SM OS 2.3 — Filtros clicables + acciones masivas

## Cambios principales

- Convierte los filtros del Centro de mando en vistas operativas.
- Agrega filtro “Mis críticos”.
- Permite selección múltiple de tareas dentro de los filtros ejecutivos.
- Agrega acciones masivas:
  - cambio de responsable,
  - cambio de estado,
  - fecha de seguimiento,
  - siguiente acción,
  - comentario masivo.
- Registra cambios masivos en el Historial como actividad del sistema.
- Agrega reporte filtrado para copiar solo el subconjunto visible.
- Mantiene modo cliente activo: solo se muestra el proyecto activo en sidebar.

## Versión

`v=2.3.0`
